Amazon Clone Project

This project is a simple Amazon-like website built using HTML, CSS, JavaScript, and a backend. The goal of this project was to understand how real e-commerce websites work, including product display, user interaction, and client-server communication.

Features

- Responsive homepage similar to Amazon  
- Product listing page  
- Product details page  
- Add to cart interface  
- Search bar for products  
- Interactive images  
- Basic user interface for login  
- Backend for handling requests  

Technologies Used-
Frontend:  
HTML  
CSS  
JavaScript  

Backend:  
for the backend also , I preferred JavaScript . In this Project javascript is the main role for both frontend and backend


What I Learned--
Through this project, I learned how to design web pages, work with JavaScript, connect frontend with backend, and understand how real e-commerce applications function.
